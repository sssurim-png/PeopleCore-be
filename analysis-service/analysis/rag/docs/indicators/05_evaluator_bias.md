# 평가자별 점수 분포 차이 진단 기준

## 분석 정의

**managerGrade** (1차 상위평가, autoGrade·calibration·finalGrade 보정 이전 원천 데이터)
기준으로, 평가자(evaluatorId, 보통 팀장·부서장) 단위 점수 분포가
사내 통상값과 얼마나 다른지 **수치적으로** 측정 + 다년 일관성 + 팀원 교체율 결합:

- 1차: 이번 시즌 분포 양 끝 또는 변별력 낮은 평가자 도출
- 2차: 도출된 후보의 과거 시즌 위치 추적
- 3차: 다년 일관성 패턴 라벨 부여
- 4차: **팀원 교체율·평가자 부서 이동** 정보 결합 (분리력 보강)

> **왜 managerGrade 인가?**
> autoGrade(자기평가 Z보정 혼합), calibration(회의 보정), finalGrade(전사 강제분포)
> 는 보정이 누적된 결과라 1차 평가자 본인 패턴이 희석됨.
> 평가자 본인의 점수 부여 패턴을 보려면 가장 원천인 managerGrade 만 사용한다.

## 핵심 지표 (수치)

### 평가자 점수 분포

| 지표 | 정의 | 단위 |
|------|------|------|
| `mean_score_evaluator` | 평가자 본인 평균 점수 | 5점 척도 |
| `sa_ratio_evaluator` | 평가자 본인 S+A 비율 | % |
| `stdev_evaluator` | 평가자 본인 점수 표준편차 (변별력 지표) | 점 |
| `mean_z` | 전사 평가자 풀 평균 대비 본인 평균의 Z-score | σ |
| `sa_z` | 전사 평가자 풀 S+A 비율 평균 대비 본인의 Z-score | σ |
| `n_evaluatees` | 평가자 본인이 평가한 사원 수 | 명 |
| `n_evaluators` | 분석 대상 시즌의 전사 평가자 풀 크기 | 명 |

### 분리력 보강 지표 (시즌 간 비교)

| 지표 | 정의 | 단위 |
|------|------|------|
| `turnover_rate` | 이전 시즌 평가 대상 사원 중 이번 시즌 빠진 비율 = 1 − (이전 ∩ 이번) ÷ 이전 | % |
| `new_evaluatee_ratio` | 이번 시즌 평가 대상 중 이전 시즌 미포함 신규 비율 | % |
| `evaluator_dept_change` | 평가자가 평가한 부서가 이전 시즌과 다른지 | bool |

## 매개변수

| 매개변수 | 기본값 | 의미 |
|---------|-------|------|
| `season_id` | 최신 | 1차 필터 대상 시즌 |
| `lookback_seasons` | 3 | 과거 추적 시즌 수 (이번 포함 시 총 4시즌까지) |
| `min_evaluatees` | 5 | 평가자 평가 최소 사원 수 (소표본 제외) |
| `min_evaluators_for_full` | 20 | 백분위 + Z-score 둘 다 적용하는 풀 크기 |
| `min_evaluators_for_zscore` | 10 | Z-score 만 적용하는 최소 풀 크기 |
| `min_evaluators_for_fallback` | 5 | fallback (상위/하위 N명 절대 도출) 최소 풀 크기 |
| `z_score_threshold` | 1.5 | Z-score 후보 도출 임계치 (±1.5σ) |
| `fallback_top_n` | 1 | fallback 모드에서 상위·하위 도출 개수 |
| `low_variance_threshold` | 0.3 | 변별력 부족 판정 표준편차 임계치 (절대) |
| `high_turnover_threshold` | 50 | "팀원 많이 바뀜" 판정 교체율 임계치 (%) |

## 분석 흐름

### 0단계: 표본 크기 자동 모드 전환

전사 평가자 풀 크기(`n_evaluators`)에 따라 분석 모드 자동 결정:

| 풀 크기 | 모드 | 1차 필터 방식 |
|--------|-----|--------------|
| ≥ 20명 | **FULL** | 백분위(상/하위 10%) + Z-score(±1.5σ) + 변별력 |
| 10~19명 | **ZSCORE** | Z-score(±1.5σ) + 변별력 |
| 5~9명 | **FALLBACK** | 상위/하위 1명 절대 도출 + 변별력 |
| < 5명 | **INSUFFICIENT** | 분석 스킵, 표본 부족 사실 명시 (단정 X) |

> 풀 크기는 **분석 시즌별로 별도 산정** — 시즌마다 평가자 수 다를 수 있음.

### 1단계: 평가자별 평균/SA비율 계산 (이번 시즌)

전사 평가자 N명 각자 평균/SA비율 계산.
모드 따라 추가 산출:
- FULL: 백분위 + Z-score
- ZSCORE: Z-score 만
- FALLBACK: 단순 순위만

### 2단계: 1차 필터 — 후보 도출 (모드별)

#### FULL 모드 (≥ 20명)
| 필터 | 임계치 |
|------|--------|
| 평균 백분위 양 끝 | mean_pctile ≥ 90 또는 ≤ 10 |
| Z-score 절대값 | |mean_z| ≥ 1.5 |
| 변별력 낮음 | stdev_evaluator ≤ 0.3 |

#### ZSCORE 모드 (10~19명)
| 필터 | 임계치 |
|------|--------|
| Z-score 절대값 | |mean_z| ≥ 1.5 |
| 변별력 낮음 | stdev_evaluator ≤ 0.3 |

#### FALLBACK 모드 (5~9명)
| 필터 | 도출 방식 |
|------|---------|
| 상위·하위 절대 N명 | 평균 점수 정렬 후 상위 1명 + 하위 1명 무조건 도출 |
| 변별력 낮음 | stdev_evaluator ≤ 0.3 (절대값 그대로) |

#### INSUFFICIENT 모드 (< 5명)
- 분석 스킵
- 결과 메시지: "평가자 풀 N=X (5명 미만), 통계 분석 의미 없음. 정성 검토 권고."

→ 어느 모드든 1차 필터 통과 = 후보

### 3단계: 후보의 과거 시즌 추적
1차 통과 후보만 과거 N시즌 동일 분석 → 다년 패턴 라벨
(과거 시즌도 표본 부족이면 해당 시즌 분석 스킵)

### 4단계: 분리력 보강 정보 결합
시즌 간 `turnover_rate`, `evaluator_dept_change` 함께 표시 → 해석 가이드

## 사내 통상 비교 — 동적 임계치

본 시스템은 절대 임계치 대신 사내 통상값을 사용 (모드별 다름):

- 비교 기준 = **그 시즌 전사 평가자 풀 분포**
- FULL/ZSCORE 모드: Z-score (전사 평균 ± 표준편차) — 풀 크기 변화에 적응
- FALLBACK 모드: 풀 자체가 작아 분포 의미 약함 → 절대 순위(상/하위 N명) 사용
- 외부 권고치 인용 X, 사내 데이터 자체가 기준
- 단, **변별력 임계치(0.3)** 와 **교체율 임계치(50%)** 는 절대값 — 명확한 사실 판정용

## 패턴 라벨 (다년 통합)

| 패턴 | 정의 |
|------|------|
| **지속 상위** | N시즌 모두 1차 필터 "상위" 도출 (FULL: mean_pctile≥90, ZSCORE: mean_z≥1.5, FALLBACK: 상위 1명) |
| **지속 하위** | N시즌 모두 1차 필터 "하위" 도출 (대칭 조건) |
| **지속 변별력 낮음** | N시즌 모두 `stdev_evaluator ≤ 0.3` |
| **변동** | 1차 필터 통과했으나 위 3가지 모두 미해당 (시즌별 위치 다름) |
| **데이터 부족** | 평가자 활동 시즌 1회 → 다년 추적 불가, 단년만 표시 |
| **표본 부족** | 분석 시즌 평가자 풀 < 5명 → 분석 자체 스킵 (단정 X, 사실 명시) |

> 라벨 = "수치 위치" 표현. 편향/관대/엄격 같은 가치판단 단어는 본문에 사용 X.

## 시그널 의미 — 모든 패턴 양쪽 가능성 + 분리력 보강

### 기본 해석 (양쪽 가능성)

| 패턴 | 가능성 1 (평가자 패턴) | 가능성 2 (외부 요인) |
|------|----------------------|---------------------|
| 지속 상위 | 평가자 후한 패턴 | 팀 일관 우수 |
| 지속 하위 | 평가자 박한 패턴 | 팀 일관 미흡 |
| 지속 변별력 낮음 | 차별 회피 (무난주의) | 팀원 실력 유사 |
| 변동 | 변덕 평가 | **팀 성과 정확 반영 (공정)** |

### 보강 정보로 해석 강도 조정

| 보강 시그널 | 해석에 미치는 영향 |
|------------|------------------|
| `turnover_rate` 낮음 (< 50%) 일관 | 팀 거의 그대로 → **분리 불가, 가능성 1·2 동등** |
| `turnover_rate` 높음 (≥ 50%) 일관 | 팀원 절반 이상 바뀌었는데 같은 패턴 → **가능성 1(평가자) 가설 강화** |
| `evaluator_dept_change = true` | 평가자가 부서 옮겼는데도 같은 패턴 → **가능성 1(평가자) 가설 추가 강화** |

→ **가설 강화 ≠ 단정.** 외부 정성 정보 (사업 KPI·조직 변화) 결합 권고는 동일.

## 권고 액션

1. 분석 결과 = 수치 차이 + 보강 정보 (즉시 처분 X, 가치판단 X)
2. 1차 필터 + 다년 패턴 + 보강 시그널 = HR 검토 안건 후보
3. 결합 분석 권고:
   - + #4 (부서별 등급분포) → 팀 인재풀 차이로 설명되는지
   - + 사업 KPI·조직 변화 정성 → 변동이 공정인지 변덕인지
4. 결합 분석에도 차이 잔존 + 보강 시그널 강함 → 평가자 패턴 가설 검토:
   - calibration 회의 우선 안건
   - 평가자 캘리브레이션 교육 검토
5. 결정은 권한자의 판단

## 표현 가이드 — 본문 vs AI 박스

본문 (수치만, 가치판단 X — 모드 명시):

- 풀 크기 충분한 경우 (FULL):
  - "전사 평가자 45명 (FULL 모드), managerGrade 기준 분포 분석"
  - "1차 필터 통과 8명 (상위 4 / 하위 3 / 변별력 낮음 1, 일부 중복)"
  - "E-021: 이번 mean=4.05 (z=+2.1, pctile 96, n=12) / 작년 (z=+1.8) / 재작년 (z=+1.9) → 패턴: 지속 상위"

- 풀 크기 작은 경우 (FALLBACK):
  - "전사 평가자 7명 (FALLBACK 모드 — 풀 작음), managerGrade 기준 분석"
  - "1차 필터: 상위 1명 + 하위 1명 절대 도출 + 변별력 낮음 0명"
  - "E-021: 이번 mean=4.05 (전사 7명 중 1위, n=12) / 작년 (1위) → 패턴: 지속 상위"

- 풀 크기 미달 (INSUFFICIENT):
  - "전사 평가자 3명 (INSUFFICIENT — 5명 미만)"
  - "본 분석은 통계적 의미 확보가 어려워 결과를 단정하지 않습니다. 정성 검토 권고."

AI 참고 박스 (1-2줄, 양쪽 가능성 + 모드 영향):

- FULL/ZSCORE 모드:
  > "E-021 의 점수 분포가 3시즌 연속 전사 평균 +1.5σ 이상입니다.
  > 평가자 본인 패턴 가설과 팀 일관 우수 가설을 데이터로 분리하기 어렵습니다.
  > 사업 성과 정성 결합을 권고합니다."

- FALLBACK 모드:
  > "E-021 은 평가자 풀이 7명으로 작아 통계 신뢰도 제한적이나,
  > 3시즌 연속 절대 1위 패턴이 관찰됩니다.
  > 정성 검토 결합을 권고합니다."

- INSUFFICIENT 모드:
  > "본 분석 시즌의 평가자 풀이 3명으로 통계 분석 의미가 확보되지 않습니다.
  > 분석 결과를 단정할 수 없어 정성 검토를 권고합니다."

## 분석 도구

- `find_evaluator_distribution_outliers(season_id, lookback_seasons, min_evaluatees, min_evaluators_for_full, min_evaluators_for_zscore, min_evaluators_for_fallback, z_score_threshold, fallback_top_n, low_variance_threshold, high_turnover_threshold)`

## 한계

1. **본질적 한계 — 평가자/팀 분리 불완전성**
   - 같은 팀을 계속 평가하는 구조에서 평가자 본인 패턴 가설과 팀 일관 성과 가설을 데이터만으로 분리 불가
   - `turnover_rate`, `evaluator_dept_change` 보강 시그널은 분리력을 일부 올릴 뿐, 완전 분리는 불가
   - → 정성 결합 검토 필수
2. **인과 단정 불가** — 모든 패턴 라벨은 양쪽 가능성 동시 존재
3. **소표본 평가자 신뢰도 낮음** — `min_evaluatees` 미만 제외, 임계치 인접 평가자는 변동 큼
4. **managerGrade 의존** — 1차 평가 데이터(별도 컬럼)가 없으면 분석 불가
5. **신규 평가자 한계** — 활동 시즌 1회만 있으면 다년 패턴 판정 X (데이터 부족 라벨)
6. **소규모 팀 turnover 노이즈** — 5~10명 팀은 1~2명 교체로도 비율 크게 변동
7. **분포 분류는 상대 개념** — 전사 평가자 풀 분포가 좁으면 작은 차이도 Z-score 가 크게 잡힐 수 있음
8. **변별력·교체율 임계치는 절대값** — 5점 척도·일반 규모 팀 가정. 다른 환경이면 재조정 필요
9. **표본 크기 의존** — 풀 < 5명 시 분석 자체 스킵. 5~9명 fallback 모드는 통계 신뢰도 제한적, "참고 자료" 수준. 10명+ 부터 Z-score 의미 시작, 20명+ 부터 백분위까지 안정. 분석 결과 활용 시 모드 명시 필수
