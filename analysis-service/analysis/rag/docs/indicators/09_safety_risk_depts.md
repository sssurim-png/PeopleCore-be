# 산업안전 위험 부서 진단 기준

## 분석 정의

근태(`CommuteRecord`) 데이터로 **한국 근로기준법 명문 한도** 위반 부서를 도출:

- **사내 통상값 비교 X** — 법규 절대 임계치 기준 (#7 워라밸과의 핵심 차이)
- 위반 부서 + 위반 사원 명단 동시 도출
- 임계 근접 부서 별도 표기 (선제 검토용)

## 적용 법규

| 위반 항목 | 법조항 | 임계치 |
|---------|------|------|
| **주 52시간 초과** | 근로기준법 §53 (기본 40h + 연장 12h) | 주 52h |
| **연속근무 한도** | 근로기준법 §55 (주휴 1일 보장) | 6일 연속 |

> 야간·휴일근무 누적은 한국법상 한도 명문 없음 (가산수당 의무만 존재).
> → 본 분석 범위 외 (워라밸은 #7, 가산수당 정합성은 별도 분석).

## 데이터 소스

| 출처 | 활용 필드 | 분석 용도 |
|------|---------|----------|
| `CommuteRecord.workDate` | 근무 일자 (월별 파티션 키) | 1주 / 연속일 집계 단위 |
| `CommuteRecord.actualWorkMinutes` | 실 근무 분 (휴게 차감 후) | 주간 합산 → 52h 비교 |
| `CommuteRecord.workStatus` | NORMAL/LATE/EARLY_LEAVE/HOLIDAY_WORK/ABSENT/AUTO_CLOSED | 출근일 판정 |
| `CommuteRecord.holidayReason` | NATIONAL/COMPANY/WEEKLY_OFF | 휴일 출근 식별 |

## 핵심 지표 (수치)

### 부서 단위

| 지표 | 정의 |
|------|------|
| `weekly_over_52h_emp_ratio` | 분석 기간 내 주 52h 초과 발생 사원 비율 |
| `weekly_over_52h_count` | 부서 내 52h 초과 발생 주 수 합계 |
| `weekly_50h_to_52h_emp_ratio` | 주 50~52h 사원 비율 (임계 근접) |
| `consecutive_violation_emp_count` | 7일 이상 연속근무 발생 사원 수 |
| `consecutive_work_days_max` | 부서 내 최대 연속근무 일수 |

### 사원 단위 (위반자 명단)

| 지표 | 정의 |
|------|------|
| `emp_violation_types` | 위반 항목 ("52h" / "연속근무" / 둘 다) |
| `emp_violation_weeks` | 52h 위반 발생 주 (예: ["2026-W08", "2026-W11"]) |
| `emp_max_weekly_hours` | 사원 분석 기간 내 최대 주간 근무시간 |
| `emp_max_consecutive_days` | 사원 분석 기간 내 최대 연속근무 일수 |

## 매개변수

| 매개변수 | 기본값 | 근거 |
|---------|-------|------|
| `weekly_hour_limit` | 52 | 한국 근로기준법 §53 |
| `max_consecutive_work_days` | 6 | 한국 근로기준법 §55 (주휴 보장) |
| `near_threshold_hours` | 50 | 임계 근접 판정 (52h 의 96%) |
| `analysis_period_months` | 3 | 집계 기간 |
| `min_dept_size` | 10 | 분석 최소 부서 인원 |

## 분석 흐름

### 1단계: 1주 단위 근무시간 집계

- **1주 = 월요일 ~ 일요일** (회사 정책 고정)
- 사원별 주간 `actualWorkMinutes` 합산 → 시간 환산
- 주간 시간 ≥ 52h → "52시간 위반 주"

### 2단계: 연속근무 일수 산출

- 사원별 `workDate` 시퀀스에서 출근일 (workStatus ≠ ABSENT 이고 휴일 아님) 연속 카운트
- 휴일(workStatus = NORMAL on holidayReason ≠ null = 휴일근무) 도 출근으로 카운트
- 7일 이상 연속 → "연속근무 위반"

### 3단계: 부서 집계 + 라벨

- 부서별 위반 사원·주 수·연속일 집계
- 라벨 부여 (아래 표)

## 부서 라벨

| 라벨 | 조건 |
|------|------|
| **법규 위반 부서** | 52h 초과 또는 7일 연속근무 1건 이상 발생 |
| **임계 근접 부서** | 위반은 없으나 주 50~52h 사원 비율 ≥ 30% |
| **통상 범위** | 위 모두 미해당 |

## 시그널 의미

법규 위반은 **객관 사실 단정 가능** (사이클성·일시성과 무관):

| 라벨 | 단정 가능 영역 | 단정 불가 영역 |
|------|--------------|--------------|
| 법규 위반 | 한도 초과 사실 | 위반의 자발/강제 원인 |
| 임계 근접 | 한도 근접 사실 | 다음 분기 위반 발생 여부 |

→ #5/#7 과 달리 위반 *사실* 자체는 단정.
→ 단, 위반 *원인* (강제 야근 vs 자발 야근) 은 여전히 데이터로 분리 X.

## 권고 액션

1. 분석 결과 = 법규 위반 부서·사원 명단 자료
2. 위반 부서 → **즉시 시정 검토 안건** (법적 리스크)
3. 위반 사원 → 개별 면담 + 근태 조정
4. 임계 근접 부서 → 선제적 부담 분산 검토
5. 결합 분석 권고:
   - + #7 (워라밸) → 사내 통상 + 법규 위반 동시 발생 시 강한 시그널
   - + 가산수당 정합성 (별도 분석) → 야간/휴일 위반 시간이 수당에 반영됐는지
6. 결정은 권한자 판단 (노무·인사위원회)

## 표현 가이드 — 본문 vs AI 박스

본문 (수치만):
- "분석 기간 2026-02 ~ 2026-04 (3개월), 분석 부서 12개 (≥10명), 1주 = 월~일"
- "법규 위반 부서: 2개 (R&D팀, 영업본부)"
- "  - R&D팀 (n=85): 52h 초과 사원 18명 (21%), 위반 주 27회, 7일 연속근무 사원 4명, 최대 연속 9일"
- "  - 영업본부 (n=42): 52h 초과 사원 6명 (14%), 위반 주 9회, 7일 연속근무 0명"
- "임계 근접 부서: 1개 (마케팅팀, 50~52h 사원 비율 33%)"
- "위반 사원 명단: ... 총 24명 (52h: 24, 연속근무: 4, 둘 다: 4)"

AI 참고 박스 (1-2줄):
- 위반 부서:
  > "R&D팀은 분석 기간 중 주 52시간 한도(근기법 §53)와 7일 연속근무 한도(§55)를
  > 모두 위반한 사원이 있습니다. 즉시 시정 검토와 사원 면담을 권고합니다."
- 임계 근접:
  > "마케팅팀은 위반은 없으나 주 50~52시간 구간 사원 비율이 33%로 임계에 근접합니다.
  > 다음 분기 위반 가능성이 있어 선제적 부담 분산 검토를 권고합니다."

## 분석 도구

## 한계

1. **한국 근로기준법 기준 한정** — 다른 국가·관할의 회사이거나 별도 사규가 더 엄격하면 본 임계치는 부족할 수 있음
2. **보호 대상 미분리** — 임산부·미성년자·연소근로자 등 추가 보호 대상에 적용되는 별도 한도(주 40h 등)는 본 분석 범위 외. 별도 분석 필요
3. **자발/강제 구분 X** — 위반 사실은 단정하나, 본인 자발 의지인지 구조적 강제인지 데이터로 분리 불가 — 면담 등 정성 결합 필요
