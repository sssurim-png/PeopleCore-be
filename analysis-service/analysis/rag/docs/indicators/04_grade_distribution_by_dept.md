# 부서별 등급 분포 진단 기준

## 분석 정의

전사 강제분포가 적용된 **finalGrade** 기준으로,
부서(deptId) 단위 S/A 등급 비율의 **편중**을 진단:

- 각 부서별 S 비율, A 비율, S+A 비율 계산
- 전사 평균과의 차이(편차) 도출
- 차이가 큰 부서 = "인재 풀 편중 부서" 후보

## 핵심 지표

| 지표 | 정의 |
|------|------|
| `s_ratio_dept` | 부서 내 S 등급 인원 ÷ 부서 전체 인원 |
| `a_ratio_dept` | 부서 내 A 등급 인원 ÷ 부서 전체 인원 |
| `sa_ratio_dept` | 부서 내 S+A 등급 인원 ÷ 부서 전체 인원 |
| `s_ratio_company` | 전사 S 비율 (강제분포 결과) |
| `delta_sa` | `sa_ratio_dept - sa_ratio_company` (단위: %p) |

## 매개변수

| 매개변수 | 기본값 | 의미 |
|---------|-------|------|
| `min_dept_size` | 10 | 최소 부서 인원 (소규모 부서 제외) |
| `delta_threshold_pp` | 10 | 전사 대비 ±%p 임계치 |
| `season_id` | 최신 | 분석 대상 시즌 |

## 사내 통상 비교 — 동적 임계치

본 시스템은 절대 임계치 대신 사내 통상값을 사용:

- 비교 기준 = **그 시즌 전사 강제분포 결과**
- 부서별 S+A 비율과 전사 S+A 비율의 차이(`delta_sa`)
- `|delta_sa| ≥ 10%p` 이면 "편중 부서" 분류
- 외부 권고치 인용 X, 사내 데이터 자체가 기준

### 분류 라벨

| 조건 | 라벨 |
|------|------|
| `delta_sa ≥ +10%p` | **상위 편중** (S/A 다수 보유) |
| `delta_sa ≤ -10%p` | **하위 편중** (S/A 소수 보유) |
| `\|delta_sa\| < 10%p` | 통상 범위 |

## 시그널 의미

이 분석이 드러내는 패턴:
- 전사 강제분포에도 불구하고 부서별 S/A 비율 차이가 큼
- "인재 풀이 두꺼운 부서" vs "얇은 부서" 가시화
- **단정 불가**: 차이의 원인은 ① 실제 인재 분포 ② 평가자 패턴 ③ 부서 특성(직무 차이) 중 어느 것인지 본 분석만으로는 구분 X
  - 평가자 패턴 분리는 **#5 (평가자 편향)** 와 결합 분석
  - 시간 안정성 검증은 **#6 (등급 변동성)** 와 결합 분석

## 권고 액션

1. 분석 결과 = 부서별 인재 분포 현황 자료 (즉시 처분 X)
2. 편중 부서 도출 → HR 검토 안건 후보
3. 결합 분석 권고:
   - + #5 → 편중 원인이 평가자 패턴인지 검증
   - + #6 → 편중이 일관된 패턴인지, 단년 현상인지 검증
   - + #2 → 하위 편중 부서에 보상 누락 우수인재가 있는지 교차 확인
4. 결정은 권한자의 판단

## 표현 가이드 — 본문 vs AI 박스

본문 (수치만):
- "전사 S+A 비율 32% (n=540)"
- "R&D팀 S+A 비율 47% (n=85, Δ +15%p)"
- "재무팀 S+A 비율 18% (n=42, Δ -14%p)"

AI 참고 박스 (1-2줄 권고):
- "R&D팀은 전사 대비 S/A 비율이 +15%p 높아 인재 풀 편중이 관찰됩니다.
  편중 원인 구분은 평가자 패턴(#5) 및 시계열 안정성(#6) 결합 분석을 권고합니다."

## 분석 도구

- `find_dept_grade_distribution(season_id, min_dept_size, delta_threshold_pp)`

## 한계

1. **인과 단정 불가** — 편중의 원인(인재/평가자/직무특성) 분리 X
2. **직무 차이 미반영** — R&D vs 영업 등 직무 본질 차이 미보정
3. **단년 시즌 분석** — 시계열 안정성은 #6 별도 분석
4. **소규모 부서 신뢰도 낮음** — `min_dept_size` 미만 제외, 임계치 인접 부서는 변동 큼
5. **finalGrade 기준** — managerGrade·autoGrade 단계의 편중은 별도 분석
